# 🔮 Judge Club

> **나의 감을 믿는 힘, 차곡차곡 쌓이는 판단의 자기확신!** 💖

Y2K 레트로 팝 & 키치 감성의 **의사결정 기록 및 피드백 다이어리** 웹 앱입니다.  
매일의 작은 판단들을 기록하고, 나중에 결과를 피드백하면서 나만의 판단 패턴을 분석할 수 있어요.

---

## ✨ 주요 기능

### 📝 일지 적기
- 날짜, 카테고리(돈·사람·커리어·건강·일상·기타), 판단 내용, 이유를 간단하게 기록
- 각 카테고리 클릭 시 간단한 설명 팝업 안내

### 📋 히스토리
- 과거 판단 기록 전체 조회
- 필터: 전체 / 최근 일주일 / 이번 달 / 결과 미입력
- 커스텀 날짜 범위 필터
- 결과 피드백 (⭕맞음 / ❌틀림 + 오답 요인 분류 + 한 줄 코멘트)
- 개별 일지 수정 / 삭제

### 💖 분석룸
- 판단 적중률 게이지 (퍼센트 표시)
- 카테고리별 적중률 랭킹 바 차트
- 오답 원인 분석 리포트
- 알고리즘 기반 개인 맞춤 조언

### 🔮 텔레파시 동기화
- **기기간 데이터 이동** (아이폰 ↔ 맥북 등)
- 8자리 랜덤 동기화 코드 발급 → 클라우드에 쏘기/가져오기
- JSON 파일 내보내기/불러오기 백업 지원

## 🔐 개인 데이터 보안 안내

> **사용자의 일지 데이터는 절대 GitHub에 저장되지 않습니다.**

- 모든 기록은 **사용자 본인의 브라우저 `localStorage`** 에만 저장됩니다.
- GitHub 리포지토리에는 **디자인과 로직 코드만** 공개되어 있습니다.
- 텔레파시 동기화 기능은 [kvdb.io](https://kvdb.io)의 **익명 공개 버킷**을 통해 동작합니다.
  - 사용자 각자의 **8자리 랜덤 코드**로 데이터가 분리 저장됩니다.
  - 개인 API 키나 계정 정보는 전혀 사용되지 않습니다.
  - 동기화 코드를 타인에게 공유하지 않으면 데이터에 접근할 수 없습니다.
